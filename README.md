# Chess.com 戰績深度統計與 AI 參考資料生成器

一個單檔 HTML 工具，抓取 Chess.com 玩家對局，產出可直接餵給 AI 模型的參考資料（含統計對比與 PGN 棋譜）。

## 功能

- **跨期對比統計**：自動以相同長度的前一期區間做對照，計算總場次、勝率、結算積分、超時/將殺/放棄勝負、和棋等變化量。
- **賽制篩選**：Bullet / Blitz / Rapid / Daily 可複選。
- **PGN 數量控制**：自選輸出 10 / 20 / 30 / 50 / 100 / 全部 局棋譜，避免一次塞給模型過多內容。
  - 統計數據一律以「全部對局」計算，僅 PGN 列表受數量限制。
- **合理的 PGN 選取策略**：
  - **代表性取樣（預設）**：依勝 / 負 / 和比例分層（最大餘數法分配名額），組內再沿時間軸均勻取樣，讓樣本貼近真實戰績分佈、涵蓋整個區間。
  - **最新優先**：取最接近結束日的 N 局，反映近期手感。
- **手機 RWD**：響應式版面、觸控友善按鈕、防 iOS 自動縮放。
- 一鍵複製或下載 `.txt`。

## 使用方式

直接以瀏覽器開啟 `index.html` 即可，無需任何建置或後端。

預設目標玩家為 `ttdragon0722`，可於 `index.html` 內的 `username` 常數修改。
